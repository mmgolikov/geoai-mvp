# Phase B Source Selection Checklist v1

## OSM / Geofabrik

- [ ] Exact GCC source file or release timestamp selected.
- [ ] Source file checksum recorded.
- [ ] Attribution text approved.
- [ ] Adapted-database/public-distribution treatment reviewed.
- [ ] AOI/filter commands tested.
- [ ] Required OSM IDs/tags retained.
- [ ] No Google/commercial tracing used.

## Overture

- [ ] Exact release ID selected through official release/STAC evidence.
- [ ] Required themes/types selected.
- [ ] Theme-level licence recorded.
- [ ] Source-level attribution generated.
- [ ] OSM attribution retained where applicable.
- [ ] Provider IDs and aliases retained.
- [ ] Public distribution treatment reviewed.

## Focus AOIs

- [ ] Marina/JBR/Palm bbox approved.
- [ ] Downtown/Business Bay/Meydan bbox approved.
- [ ] Dubai South/Jebel Ali bbox approved.
- [ ] Creek/DXB optional scope decision recorded.
- [ ] Processing envelope clearly marked non-official.

## Initial geometry

- [ ] Transport source selected.
- [ ] Anchor source selected.
- [ ] Building source precedence selected.
- [ ] Land-use/water source selected.
- [ ] Construction target selection method approved.
- [ ] Selected AOI derivation rules approved.
- [ ] Community/context boundary layer held until manual plausibility review.

## Asset storage

- [ ] Public repository size policy approved.
- [ ] Large geometry storage/distribution path approved.
- [ ] Raw source files excluded unless explicitly allowed.
- [ ] Manifests and checksums committed.
- [ ] Confidential/licensed/official data excluded.

## Release eligibility

No B1 prompt is issued until every required item above is complete or documented as a blocking condition in the B1 Change Request.

## Required caveat

**Screening hypothesis; official validation required; not a legal, cadastral, zoning, planning or valuation conclusion.**

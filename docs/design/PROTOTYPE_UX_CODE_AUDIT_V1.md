# Prototype UX Code Audit v1

Status: design input.

This audit records UX rules encoded in the working prototype.

Key rules:

- workspace route uses top navigation plus workspace shell
- workspace height is viewport based below the top navigation
- desktop workspace uses main canvas plus right command panel
- right command panel width is about 380px
- main canvas can show map, report preview, comparison dashboard or analysis dashboard
- right panel remains the control surface
- right panel has internal scroll
- right panel footer contains Add to compare and primary action
- analysis dashboard first view is compact and viewport bound
- comparison dashboard follows the same first view contract
- report preview is a workspace state

Design consequence:

The next Figma pass should preserve this UX and improve only visual quality.

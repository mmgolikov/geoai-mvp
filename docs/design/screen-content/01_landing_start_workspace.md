# Screen Content Spec — Landing, Start, Workspace

Status: draft / content-first design input

## Universal screen contract

Each screen must contain:

1. Context
2. Evidence
3. Decision
4. Action
5. Output

No card should exist unless it supports one of these layers.

## Landing V6

Goal: explain GeoAI as spatial decision intelligence in under 10 seconds.

Primary audience: investor, developer, bank, public-sector sponsor.

Primary action: Open demo workspace.

Required content blocks:

- Hero: headline, value proposition, UAE wedge, primary CTA, secondary CTA.
- Product preview: Start, Workspace, Evidence, Analysis, Report.
- Decision pipeline: Start, Select, Evidence, Analyze, Compare, Report.
- Use cases: development, construction, portfolio, government monitoring, climate, digital twin.
- Evidence trust: demo, open, uploaded, reviewed, source status.
- Pilot CTA: raise, pilots, Dubai/Abu Dhabi focus, deliverable output.

Do not show:

- Generic SaaS filler.
- Decorative map without decision content.
- Claims that imply connected or approved data before review.

## Start / Scenario Hub

Goal: define the decision before map interaction.

Primary action: Continue to Workspace.

Required content blocks:

- Project context: active project, geography, client type.
- Audience selector: B2B, B2G, portfolio, custom.
- Role selector: developer, fund, bank, authority, consultant.
- Scenario cards: site screening, construction, portfolio, government, climate, digital twin.
- Mode selector: map-first or criteria-first.
- Output preview: score, evidence gaps, memo/report output.

Do not show:

- Long settings form on first view.
- Admin readiness.
- Source registry details.

## Workspace / Selected Target

Goal: select and understand the spatial target.

Primary action: Run analysis.

Required content blocks:

- Map canvas: basemap, selected target, layers, candidate geometry.
- Target card: name, coordinates, area, type, source mode.
- Source strip: demo, open, uploaded, review states.
- Layer summary: land, transport, prices, climate, zoning, construction.
- Review preview: parcel, title, zoning, FAR, utilities.
- Action footer: run analysis, save target, open evidence.

Do not show:

- Report builder before analysis.
- Data room details.
- Full admin/source registry.

# Compare and Report Screen Content Spec

Status: draft.

## Compare

The Compare screen needs:

- candidate cards
- score comparison
- evidence state per candidate
- short recommendation
- export action

Design note: avoid wide dense tables on mobile.

## Report

The Report screen needs:

- report preview
- section checklist
- source appendix
- caveat block
- export actions

Design note: report output must be visible as a primary product result.

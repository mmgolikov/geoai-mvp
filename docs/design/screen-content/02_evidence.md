# Screen Content Spec — Evidence

Status: draft / content-first design input

## Evidence / Source Confidence

Goal: show what supports the analysis and what is still missing.

Primary action: Review evidence.

Required content blocks:

- Evidence confidence summary.
- Source rows with name, type, mode, confidence, note and date.
- Uploaded context with file name, type, status and review state.
- Review gaps for parcel, zoning, title, market and utility evidence.
- Output guardrail that explains what the result can be used for.

Required source states:

- Demo
- Open
- Uploaded
- Reviewed
- Review required

Do not show:

- Hidden notes only in a tooltip.
- A score or recommendation without an evidence state.
